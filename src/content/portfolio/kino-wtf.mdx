### About
[kino.wtf](https://www.kino.wtf) is a a collection of daily/weekly games related to movies.

### How to play
- Use the hints provided to guess a film or person.
- If you guess incorrectly, another hint and/or piece of trivia will be revealed.
- You have 6 attempts to guess the answer.

### Credits
- I curated the list of actors, directors and movies and update the list periodically. Find the lists on my [IMDb page](https://www.imdb.com/user/ur82737568/lists/?ref_=hm_nv_lst). 
- Metadata needed for each game scraped using the [TMDb API](https://developer.themoviedb.org/reference/intro/getting-started).
- Inspired by [Wordle](https://www.nytimes.com/games/wordle/index.html), [Framed](https://framed.wtf) and [all the other spinoffs](https://rwmpelstilzchen.gitlab.io/wordles/).

### Tools used
- Built using [React](https://react.dev), [Tailwind](https://tailwindcss.com) and [Daisy UI](https://daisyui.com/).
- Deployed on [vercel](https://vercel.app).

[GitHub](https://github.com/kaischuygon/kino.wtf)
